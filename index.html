CSS Snippet Cheatsheet


Do you have trouble remembering CSS syntax? This is the place to find it. Place further hard to remember syntax on this page to increase the knowledge bank. Then just click on the syntax and it will automatically be highlighted for you. How cool is that?!

Flexible Row

Use this to create a Flexbox row layout

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
    }

Flexible Row

Use this to create a Flexbox column layout

.column {
    display: flex;
    flex-direction: column;
    }

CSS Grid Layout

Use the below to create a 12 column gird from CSS

.grid {
    display: grid;
    width: 100%;
    grid-template-columns: repeat(12, 1fr);
    }

Linear Gradients

Create a linear background gradient from top to bottom

.linear-gradient-background {
    background-image: linear-gradient (
    rgba(232, 102, 236, 0.3) 0%,
    rgba(232, 102, 236, 0.6) 100%
    );
    }

Box Transition Glow

Create a glow on hover using transition and box-shadow

.code-card .card-header {
    border-radius: 8px;
    transition: all 0.5s ease-in-out;
    }

    .code-card:hover,
    .code-card:hover .card-header {
    box-shadow: inset 0px 0px 8px rgba (232, 102, 236, 1), 0 0 15px rgba (232, 102, 236, 1);
    }

Overlay Card with Title

Position properties and negative margins can raise elements above natural starting point

.card-header {
    position: relative;
    margin-top: -20px;
    }
